2. 먹이 << end
	- 생성:	게임이 시작되면 최대 150개까지 먹이가 생성된다. (딜레이 0.25초에 한 번 1/4 확률로 생성된다.)
	- 소멸:	생성되고 일정 시간이 지난 먹이는 사라진다.
	- 충돌:	주인공세포/바이러스와 충돌하면 사라지고 충돌한 세포/바이러스의 크기가 커진다.

3. 바이러스 << continue
	- 생성:	게임이 시작되면 임의의 시간마다 랜덤한 위치에 처음 주인공세포의 크기로 생성된다. (최대 5개)
	- 이동:	자유롭게 이동하다가 주인공 세포와 가까워지면 주인공 세포쪽으로 이동한다.
	- 분열:	일정 크기가 되면 분열한다.
	- 합체:	분열된지 일정 시간이 지나면 합쳐진다.
	- 소멸:	주인공 세포보다 크기가 작아 주인공 세포에게 먹히면 소멸한다.
	- 충돌:	주인공 세포보다 큰 바이러스와 충돌했을 때 게임이 종료된다.

4. 트랩
	- 생성: 	게임이 시작되면 임의의 개수로 랜덤한 위치에 트랩이 생성된다. 시간에 따라 추가로 생성된다. (최대 7개)
		생성될 때 애니메이션 > 작은 크기에서 점점 커진다.
		
	- 이동: 	자유롭게 이동한다.
	- 충돌: 	주인공세포/바이러스와 충돌했을 때, 트랩의 크기가 크면 세포/바이러스가 분열한다.
	- 소멸:	주인공세포/바이러스와 충돌하여 분열시키면 소멸한다.

5. 점보
	- 생성: 	게임이 시작되면 랜덤한 위치에 점보 아이템이 생성된다.
	- 이동: 	자유롭게 이동하다가 주인공 세포와 가까워지면 빠른 속도로 이동하여 주인공과 부딪힌다.
	- 충돌: 	주인공 세포와 부딪히면 주인공세포가 삼각형 모양으로 바뀌고 주변 일정 영역 안 먹이를 모두 먹는다. 
		주인공 세포의 속도가 더 느려진다.
		일정 시간 후 주인공 세포로 돌아가고 점보아이템은 분리되어 자유롭게 이동한다.
		일정 시간동안 점보 아이템을 먹을 수 없다.

6. 카메라 << end
	- 카메라 이동 모드:		메뉴 View-Follow Player / 
				카메라가 이동하며 주인공 세포를 중앙에 그린다. 
				맵의 가장자리로 가면 세포가 중앙에서 벗어나 움직인다.
	- 카메라 고정 모드:		메뉴 View-Fixed Map / 
				맵의 전체를 탑뷰로 보여준다.

7. 게임UI << end
	- 메인 화면:	게임 시작
	- 게임 진행:	메뉴에서 설정을 바꾸거나 게임 리셋/종료/일시정지/이어서플레이를 할 수 있고 마우스로 게임을 플레이한다.
	- 결과 화면:	먹은 먹이 수, 세포의 크기(반지름), 플레이시간 출력. 재시도/게임종료 선택

8. 게임 명령어(메뉴) << end
	- Game-Start:	게임 시작 | 메인화면 -> 플레이화면 (플레이 중일 때 누르면 작동x) 
	- Game-Pause/C···:	게임 일시정지/이어서 플레이 
	- Game-Reset:	게임 리셋 | 메인화면으로 돌아간다. 처음부터 다시 시작할 수 있다. 
	- Game-Exit:	게임 종료

	- View-Follow Player:	카메라 이동 모드
	- View-Fixed Map:		카메라 고정 모드

	- HUD Info Display-On:	주인공 세포의 크기, 먹은 먹이 수, 플레이 시간 출력 << 플레이 시간 출력 구현해야함
